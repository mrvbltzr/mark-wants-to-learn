Content Box: Total width is 200px (content) + 20px (left padding) + 20px (right padding) + 5px (left border) + 5px (right border) = 250px.
Border Box: Total width is 200px (including padding and border). Content width is actually 200px - 20px (left padding) - 20px (right padding) - 5px (left border) - 5px (right border) = 150px.
